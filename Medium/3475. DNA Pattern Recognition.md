## 🚀 3475. DNA Pattern Recognition

### 💡 Idea

The goal is to identify whether each DNA sequence contains the following patterns:

- Starts with **`ATG`** (start codon)
- Ends with **`TAA`**, **`TAG`**, or **`TGA`** (stop codons)
- Contains the motif **`ATAT`**
- Contains at least **3 consecutive `G`** characters

We use a combination of `LIKE` and `REGEXP` operators to check each pattern and return `1` if the pattern exists, otherwise `0`. :contentReference[oaicite:0]{index=0}

---

## 💻 Solution

```sql
SELECT *,
dna_sequence LIKE 'ATG%' AS has_start,
dna_sequence REGEXP 'TAA$|TAG$|TGA$' AS has_stop,
dna_sequence LIKE '%ATAT%' AS has_atat,
dna_sequence REGEXP 'GGG+' AS has_ggg
FROM Samples
ORDER BY sample_id;
```

---

### 🛠️ How It Works

1. Use `LIKE 'ATG%'` to check if the DNA sequence starts with `ATG`.
2. Use `REGEXP 'TAA$|TAG$|TGA$'` to check if the sequence ends with any valid stop codon.
3. Use `LIKE '%ATAT%'` to check if the sequence contains the motif `ATAT`.
4. Use `REGEXP 'GGG+'` to check if the sequence contains **3 or more consecutive `G`** characters.
5. Return all samples ordered by `sample_id`.

---

### ⏱️ Complexity

**Time Complexity:** `O(n × m)`

- `n` = number of rows
- `m` = average DNA sequence length

**Space Complexity:** `O(1)`
