## 🚀 3436. Find Valid Emails

### 💡 Idea

The goal is to find all users whose email addresses follow the required format.

We use the `REGEXP` operator to validate the email pattern by checking the username, domain name, and domain extension.

---

## 💻 Solution

```sql
SELECT
    user_id,
    email
FROM Users
WHERE email REGEXP '^[A-Za-z0-9_]+@[A-Za-z]+\\.com$'
ORDER BY user_id;
```

---

### 🛠️ How It Works

1. Use the `REGEXP` operator to filter only the email addresses that match the required pattern.
2. Match the username using `[A-Za-z0-9_]+`, allowing only letters, digits, and underscores before the `@`.
3. Match the domain name using `[A-Za-z]+`, ensuring it contains only letters.
4. Match the literal `.com` extension using `\\.com`.
5. Return the valid email addresses ordered by `user_id`.

---

### 🔄 Other Approaches

- ✅ `REGEXP` *(recommended approach)*
- ✅ Multiple `LIKE()` and string functions *(more complex and less efficient)*

---

### ⏱️ Complexity

**Time Complexity:** `O(n × m)`

*(`n` = number of rows, `m` = average email length)*

**Space Complexity:** `O(1)`
